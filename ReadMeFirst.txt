2d.Scrollax — Installation Instructions
========================================

Thanks for downloading 2d.Scrollax!

WHY THE SECURITY WARNING SHOWS UP
----------------------------------
This app is not notarized by Apple. Apple's Developer Program has country
eligibility requirements that don't match mine, so I'm not able to enroll
in their $99/year Developer ID program to get this app verified. That is
the only reason macOS shows a warning when you open it — it isn't a sign
that anything is wrong with the app.

Apple's notarization process exists to check whether an app is safe to
run. Since this project is fully open source, you (or anyone else) can
read the actual code yourself before running it — there's nothing hidden,
so there's no real question about safety here.

HOW TO INSTALL
---------------
1. Double-click the 2d.Scrollax.dmg file you downloaded to open it.
2. Drag 2d.Scrollax.app into the Applications folder shown in this window.
3. Open your Applications folder and double-click 2d.Scrollax to launch it.
4. macOS will show a warning that it can't verify the developer — don't
   worry, this is expected.
5. Open System Settings.
6. Go to Privacy & Security.
7. Scroll all the way to the bottom.
8. Click "Open Anyway".
9. Launch the app again — it will now open without Apple's verification.

Enjoy!

Thank you for your support.
